# Upload to GitHub

The project is ready to push. Remote is already set to **d3connect2/gmail-spam-analyzer**.

## 1. Create the repo on GitHub (one time)

1. **Sign in** at https://github.com
2. Open: **https://github.com/new?name=gmail-spam-analyzer**
3. Leave **Repository name** as `gmail-spam-analyzer` (or change it)
4. Choose **Public**
5. **Do not** add README, .gitignore, or license
6. Click **Create repository**

## 2. Push from your machine

In a terminal:

```bash
cd /Users/soufiane/Documents/projects/gmail-spam-analyzer
git push -u origin main
```

After that, the project will be at **https://github.com/d3connect2/gmail-spam-analyzer**.
